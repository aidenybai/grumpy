Examples

Importing wear.js

Basically requiring the module and constructing the class.


const Wear = require('wear');
const group = new Wear();

Declaring and deleting pairs

These are inherited from maps, you can do these in groups too.


group.set('user', 'John Doe');
 
group.get('user'); // returns 'John Doe'

Converting to Array

There are two ways you might want to convert a group into an Array. The first way is the array or keyArray methods. They simply create an array from the items in the group, but also caches it too:


group.set('user1', 'John Doe');
group.set('user2', 'Mary Jane');
group.set('user3', 'Joe Schmoe');
 
group.array(); // returns ['John Doe', 'Mary Jane', 'Joe Schmoe']
group.keyArray(); // returns ['user1', 'user2', 'user3']

Getting first, last, and random keys/values

Great utility functions without needing to rewrite code over and over. The methods have a 'count' parameter to specify how many keys/values you want to fetch.


group.set('user1', 'John Doe');
group.set('user2', 'Mary Jane');
group.set('user3', 'Joe Schmoe');

group.first(); // returns 'John Doe'
group.firstKey(); // returns 'user1'

group.last(); // returns 'Joe Schmoe'
group.lastKey(); // returns 'user3'

group.random(); // returns 'John Doe', 'Mary Jane', or 'Joe Schmoe'
group.randomKey(); // returns 'user1', 'user2', or 'user3'

Concatinating and cloning

Easily combine and create new groups.


group1.set('user1', 'John Doe');
group2.set('user2', 'Mary Jane');
group3.set('user3', 'Joe Schmoe');

const group = group1.concat(group2, group3); // returns group {'user1' => 'John Doe', 'user2' => 'Mary Jane', 'user3' => 'Joe Schmoe'}
const clonedGroup = group.clone();

Manipulating groups

Many of the methods on group are based on their namesake in Array.


group.set('user1', 'John Doe');
group.set('user2', 'Mary Jane');
group.set('user3', 'Joe Schmoe');

group.sift(user => user === 'John Doe'); // removes 'Mary Jane' and 'Joe Schmoe' from the group
group.filter(user => wanted.includes(user)); // returns Group {'user1' => 'John Doe'}
group.map(user => user).join(', '); // returns 'John Doe, Mary Jane, Joe Schmoe'
group.sort(); // returns Group {'user3' => 'Joe Schmoe', 'user1' => 'John Doe', 'user2' => 'Mary Jane'}

group.some(user => 'John Doe'.includes(user)); // returns true
group.every(user => user instanceof String); // returns true

group.each(user => console.log(user)); // logs each name in the console
group.cut(user => user.startsWith('J')); // (1): group {'user1' => 'John Doe', 'user3' => 'Joe Schmoe'}; (2): group {'user2' => 'Mary Jane'}
