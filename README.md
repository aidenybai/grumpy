## Features

* 🎉 Small file size (~1kb)  
* ⚡️ Blazing fast performance  
* 🚀 Minimal & intuitive API

## Installation

NodeJS
```bash
$ yarn add grumpy
```

Browser
```html
<script src="http://unpkg.com/grumpy/dist/index.min.js"></script>
```

## Usage

```js
import Grumpy from 'grumpy';
const group = new Grumpy();

group.set('key', 'value');

group.get('key'); // returns 'value'
```
